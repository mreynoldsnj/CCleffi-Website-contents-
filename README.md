# CCleffi-Website-contents
CDG Real Estate Watch Repository Instructions

Purpose

This repository contains static, data-driven real estate reports generated by CDG Digital Media.

Required Output Format

When generating reports:

* Always create production-ready HTML files.
* Use responsive design.
* Embed CSS directly into the document unless otherwise requested.
* Include CDG Digital Media branding.
* Include data source citations.
* Include report generation date.
* Include downloadable chart visualizations rendered in SVG.
* Avoid placeholder text.

Deliverables

For each report create:

1. index.html
2. report.json (source data)
3. charts.svg
4. README.md summary

GitHub Pages Compatibility

All output must be compatible with GitHub Pages static hosting.

Real Estate Data Standards

Reports should include:

* Median sale price
* Median listing price
* Price per square foot
* Inventory trend
* Days on market
* Sales volume
* Rental market metrics
* 90-day trend analysis
* Year-over-year comparison

Quality Requirements

* Mobile responsive
* No broken links
* Accessible HTML
* SEO metadata included
* Sources cited
* Timestamp included

Preferred Output

Whenever possible generate complete deployable files rather than code snippets.
